# Cascade — Monetization Model

## Revenue Streams

### 1. Trading Rake
**Primary revenue source**

- Small fee on every bet (1-3%)
- Taken from winning payouts
- Scales directly with platform volume

**Example:**
- User bets 100 CAS, wins 200 CAS
- Platform takes 2-6 CAS rake
- User receives 194-198 CAS

### 2. Module Royalties (Platform Cut)
**Secondary revenue**

When a module is used in chains:
- Creator earns royalty (e.g., 0.5% of bets on chains containing their module)
- Platform takes cut of royalty (e.g., 20% of creator royalty)

**Incentive alignment:** Quality modules get reused → creators earn → platform earns

### 3. Token Sales
**Initial liquidity**

- Users buy CAS tokens with fiat (via Stripe) or Bitcoin/Lightning
- Platform earns spread on token sales
- Or: charge flat fee for token purchases

### 4. Premium Tiers (Future)
**Enterprise/Power User**

- API access for automated trading
- Bulk data exports
- Custom analytics
- White-label for organizations
- Agent hosting/management

### 5. Sponsored Modules (Future)
**B2B revenue**

- Organizations pay to create high-visibility modules
- "Official" modules for company events (earnings, product launches)
- Clearly marked as sponsored

## Unit Economics (Target)

- **Average bet size:** 50 CAS
- **Average rake:** 2% = 1 CAS per bet
- **Bets per user per month:** 20
- **Revenue per user per month:** 20 CAS

At 100k users:
- 2M bets/month
- 2M CAS rake/month
- Plus royalty cuts, token sales, premium

## Token Economics

**CAS Token:**
- Internal unit of account
- 1 CAS = $0.01 (or pegged to sats)
- Cashu mint backs tokens

**No speculation on token itself** — CAS is utility, not investment.

## Regulatory Considerations

- Real-money prediction markets = gambling regulations in many jurisdictions
- Potential approaches:
  - Geo-restrict to friendly jurisdictions
  - Operate as "information markets" (research exemption)
  - Decentralize sufficiently to avoid single operator
  - Start with play money, add real money later

## Break-even Analysis

**Fixed costs (monthly):**
- Infrastructure: $2,000
- Team (small): $20,000
- Total: ~$25,000

**Break-even at 2% rake:**
- Need 1.25M CAS in bets/month
- = 25,000 bets at 50 CAS average
- = ~1,250 active users betting 20x/month
