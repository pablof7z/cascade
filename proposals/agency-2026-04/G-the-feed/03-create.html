Step 04 · Seed · hold to publish

Decisions · create flow

Publish = first trade. There's no "publish draft then place first order." The publish gesture IS the opening position. One hold, one commit, one record.
Category is detected, not chosen. As the claim is typed, the category and obvious tags auto-fill. The user can edit, but the default path has zero taxonomy friction.
Similar markets surface early. Before the user writes a case, we show three adjacent claims already trading. They can link, differentiate, or change course.
Seed picker feels like a trade. The same side-toggle + amount pad + quote strip + hold-button as every other Cascade trade. Users never learn a second interaction for publishing.
The whole flow is one drag away from cancel. Any step can be swiped down to minimize to a "draft" chip in the feed. No modal dead-ends. No losing work.
