## TikTok: 60-Second Scenarios

**Format:** Quick, punchy scenario breakdowns.

**Content formula:**
```
Hook (3s): "If [current event], here's what happens next..."
Setup (15s): Explain the thesis
Branches (30s): Walk through 2-3 scenarios with probabilities
CTA (10s): "What do you think? Drop your prediction."
```

**Example scripts:**

**#1: "What if OpenAI releases GPT-5 this month?"**
- Hook: "GPT-5 might drop any day. Here's the cascade..."
- Branch 1: Google accelerates Gemini (70%)
- Branch 2: AI regulation fast-tracked (40%)
- Branch 3: Coding jobs disrupted faster (85%)
- CTA: "Which scenario are you betting on?"

**#2: "What if Bitcoin hits $200K?"**
- Hook: "Everyone's watching Bitcoin. But what happens AFTER 200K?"
- Branch 1: Altcoin season (80%)
- Branch 2: Regulatory crackdown (50%)
- Branch 3: Institutional FOMO wave (75%)

**Production:**
- Simple graphics (scenario trees)
- Voiceover (Pablo or AI voice)
- Text overlays
- Trending sounds where appropriate

**Posting schedule:** Mon/Wed/Fri, 6pm EST
